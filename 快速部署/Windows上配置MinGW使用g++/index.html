    MinGW的下载安装

    注意：这里我们下载的是GNU软件管家，用来下载我们真正需要的g++程序 - 首先下载管家Download mingw-get-setup.exe (86.5 kB)，打开后等待5秒后开始下载

    • 安装流程安装，注意安装目录，默认的地址可以避免未知问题

    • 等待安装后continue

    • 之后打开界面，来安装我们需要的程序。选择红框中的模块，小白建议全选

    • 点击前面的方框进行选择
    • 为了方便其他编辑器使用g++来配置，顺便选择红框中的这几个，如果不需要也可以跳过
    • 之后按左上角Installationt->Apply Changes进行安装（如果下载不了试试翻墙）
    • 安装完成后变成绿色的方格即为安装成功

    环境变量的配置

    为什么要配置环境变量？

    • 如果我们想在命令行中使用g++等命令，有两种方法：

      • cd进入到可执行程序.exe所在目录，执行命令
      • 配置环境变量，在任何目录都可以执行命令

    • 这样看来明显后者更符合我们的实际需求。如果不配置环境变量的话，一般会提示不是内部或外部命令，也不是可运行的程序或批处理文件

    • 配置环境变量后，系统会在环境变量的所有路径里查找相关的应用程序，我们也就能够随时随地使用了。

    配置g++环境变量

    • 我的电脑->属性->高级系统设置->环境变量
    • 点击新建编辑如图，我的不是默认的路径，填入你的MinGW安装的目录，一般是C:\MinGW\bin，为了方便以后配置更多信息，这里分两步，先配置一个根目录，再配置bin目录。百分号的意思是引用该变量的值

    • 最后配置下Path值，其他的都是变量，系统只找Path路径里的文件。多个路径之间要用英文的分号隔开。
    • OK，到此配置完成，按win+R输入cmd打开命令行，输入g++ -v后回车,如果一切顺利的话，输出如下

    测试编译

    • 接下来我们来试试g++能否顺利编译，在桌面新建文件夹，在里面编写test.cpp文件如下
    #include<iostream>
#include<stdlib.h>      // system函数需要引入
using namespace std;
int main(){
    cout<<"Hello World!"<<endl;
    return 0;
    system("pause");    // 表示按任意键退出
}
    • 拷贝下test.cpp文件所在路径，在命令行中cd进入，然后执行编译g++ test.cpp，顺利的话会生成a.exe，运行后输出Hello World。
    • 如果报错的话，一般是文件引入出错，因为我们没有配置C++的包含文件所在路径。为了使环境变量方便管理，分两步配置，先配置变量，再在Path中引入，下面是参考命名和相关路径
      • 头文件(MINGW_HEAD) %MINGW%\lib\gcc\mingw32\5.3.0\include\c++;%MINGW%\lib\gcc\mingw32\5.3.0\include\c++\mingw32;%MINGW%\lib\gcc\mingw32\5.3.0\include\c++\backward;%MINGW%\lib\gcc\mingw32\5.3.0\include;%MINGW%/include;%MINGW%/lib/gcc/mingw32/5.3.0/include
      • 库文件(MINGW_LIB) %MINGW%/lib;%MINGW%/lib/gcc/mingw32/5.3.0
    • 版本不同的需要做下修改，此处是5.3.0。
    • 配置环境变量后需要重启cmd

    到此配置结束，g++收入囊中
