    TP5集成阿里大于

    • 阿里大于-快速开始
    • 阿里大于-API

    • TP5-扩展类库

      1. 根据快速开始介绍的流程进行接入，注册后先申请模板和签名等待审核（快的话几分钟）
      2. 下载相应语言的SDK，放入TP5工程的extend目录下。（先解压，留下里面的压缩包，删掉其他，再解压，删掉压缩包，将文件改名为alidayu方便导入）
      3. 参考TP5扩展类库，在需要使用阿里大于的地方进行文件导入。参考代码：
        （此时扩展类库的目录纵深层次为extend/alidayu/top/request/）
        use think\Loader; //导入第三方类库 Loader::import( "alidayu.TopSdk", EXTEND_PATH); //导入需要用到的类的文件 Loader::import('alidayu.top.TopClient', EXTEND_PATH); //导入发短信的类文件 Loader::import('alidayu.top.request.AlibabaAliqinFcSmsNumSendRequest', EXTEND_PATH);
      4. 实例化类时，同样参考扩展类库的介绍，一下是具体实例:
        ``` $c = new \TopClient(); //请填写自己的app key $c->appkey = config("APP_KEY"); //请填写自己的app secret $c->secretKey = config("APP_SECRET_KEY"); $req = new \AlibabaAliqinFcSmsNumSendRequest(); $req->setExtend("123456"); $req->setSmsType("normal"); $req->setSmsFreeSignName(config('SMS_FREE_SIGN_NAME')); $my_code = $data['code']; $req->setSmsParam("{\"code\": \"$my_code\"}"); //请填写需要接收的手机号码 $req->setRecNum($u_Phone); //短信模板id $req->setSmsTemplateCode(config('SMS_TEMPLATE_CODE'));

      $resp = $c->execute($req);

      // print_r($resp); if($resp->code == 0){ $mes['message'] = '验证码发送成功'; return $this->showSuc($mes); }else{ return $this->showError('验证码发送失败'); } ```
