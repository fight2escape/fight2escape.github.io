    快速上手phpredis

    Redis 是一个高性能的key-value数据库。 redis的出现，很大程度补偿了memcached这类key/value存储的不足，在部 分场合可以对关系数据库起到很好的补充作用。它提供了Java，C/C++，C#，PHP，JavaScript，Perl，Object-C，Python，Ruby，Erlang等客户端，使用很方便。

    安装redis

    • http://www.redis.cn/ #获取下载链接
    • wget http://download.redis.io/releases/redis-3.2.9.tar.gz #下载
    • yum -y install gcc tcl #依赖的程序
    • tar -xf redis-3.2.9.tar.gz #解压
    • cd redis-3.2.9 #进入
    • make #编译
    • sudo make install #安装
    • mkdir -p /home/vagrant/config/redis/ #用来存放配置文件
    • cp redis.conf /home/vagrant/config/redis/redis.conf #安装包中的redis.conf拷贝到指定位置，该配置文件用来启动redis
    • vim /home/vagrant/config/redis/redis.conf #修改配置文件
      • daemonize yes #允许后台运行
      • port 7200 #默认6379，为了多实例或安全性问题
    • redis-server /home/vagrant/config/redis/redis.conf #启动redis，如果是非root用户，需要指定绝对路径sudo /usr/local/bin/redis-server /home/vagrant/config/redis/redis.conf
    • ps aux|grep redis-server #查看redis进程

    安装phpredis（php的redis扩展）

    • 提前安装好php，以及phpize和php-config
      • yum -y install php-devel #（也可能是php70w）直接安装php的开发工具包，其中就包含了所需的两个工具
      • which phpize
      • which php-config #记录下两者的安装路径，一般在/usr/bin/下
    • wget https://github.com/phpredis/phpredis/archive/develop.zip #phpredis扩展下载
    • unzip develop.zip #解压
    • cd php-develop #进入该文件夹
    • phpize #用来生成 configure配置文件
    • ./configure --with-php-config=/usr/bin/php-config #php-config存放所有本地的php配置，添加这个参数的最大好处在于，后面make install 时可以直接将编译生成好的.so文件直接安装在你需要的地方，否则的话你就需要手动copy
    • make #编译
    • make install #安装需要root权限，非root用户请sudo
    • php --ini #查看php的配置文件所在位置
    • vim /etc/php.ini #修改配置文件
    • extension=redis.so #在文件最后面加上
    • php -m #查看所有php扩展，看redis是否在其中
    • systemctl restart php-fpm #最后记得重启下php-fpm
