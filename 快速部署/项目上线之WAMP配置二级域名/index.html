    项目上线之WAMP配置二级域名

    • 主要内容如下：
    • 原理

    • 配置方法

    • 字数：300+

    原理

    • （回顾）将需要的二级域名解析。如腾讯云，云解析中，选择主域名，然后添加记录（二级域名），A类型，如果主域名是badwolfs.cn，要添加的二级域名是bbs.badwolfs.cn，在记录中只写bbs，之后填写服务器IP，保存后十分钟内生效。

    • 具体原理参见《项目上线之LAMP配置二级域名》。

    配置方法

    1) 先打开虚拟域名模块。 - 在httpd.conf中，搜索vhost - 找到模块，去掉前面的注释#

    LoadModule vhost_alias_module modules/mod_vhost_alias.so
Include conf/extra/httpd-vhosts.conf

    2) 打开httpd-vhosts.conf文件，根据模板添加新的虚拟域名。如 

    <VirtualHost *:80>
    ServerAdmin webmaster@dummy-host.example.com
    DocumentRoot "C:/Program Files (x86)/wamp/www/newthink/public/"     // 这里是你的站点入口文件所在路径
    ServerName nt.badwolfs.cn   // 这是你想要设置的二级域名
    #ServerAlias www.dummy-host.example.com
    ErrorLog "logs/dummy-host.example.com-error.log"
    CustomLog "logs/dummy-host.example.com-access.log" common
</VirtualHost>

    3) 接着，打开C:\Windows\System32\drivers\etc\hosts文件，根据模板添加，如

    127.0.0.1       nt.badwolfs.cn

    4) 之后重启WAMP，差不多了，不行再等等，可能是域名解析未完成。如果超过十分钟后还不行，检查代码，谷歌解决。
