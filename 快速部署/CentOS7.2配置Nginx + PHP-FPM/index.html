    CentOS7.2配置Nginx + PHP-FPM

    • 主要内容如下：
    • 安装Nginx
    • 安装PHP7
    • 安装php-fpm

    • 更改nginx配置文件识别php

    • 字数：500+

    从LAMP转到LNMP的同学可能会困惑这个fpm是什么鬼，为什么要装这个！

    • 我这里简单解释下，Apache和Nginx都是web服务器，本身不能处理业务逻辑，我们需要通过php来执行。
    • 在Apache中，因为内嵌了php模块，意思是装了Apache和php后你就不需要做额外的配置，只要模块没被关掉，默认就能用了。

    • 但在Nginx中，我们需要手动来进行配置，Nginx才能和php进行通信，这里我们就需要php-fpm这个“工具”和Nginx进行交互。

    • PS：

      • Nginx配置文件所在路径 /etc/nginx/下的nginx.conf和conf.d下面的*.conf
      • Nginx站点根目录在/usr/share/nginx/html

    安装配置Nginx

    • 安装
    yum install yum-priorities -y  
#wget http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm  
rpm -ivh nginx-release-centos-7-0.el7.ngx.noarch.rpm  
yum install nginx  

systemctl start nginx.service      #启动nginx  
systemctl stop nginx.service       #停止  
systemctl restart nginx.service    #重启  
systemctl enable nginx.service     #设置开机启动
    • 更改Nginx的端口号（这里可不改）
    cd /etc/nginx/conf.d/             #配置文件所在目录          
vim default.conf                  #打开配置文件 
                                  #把listen 80修改后 
:wq!                              #保存退出
systemctl restart nginx.service   #重启nginx

    安装PHP7

    yum -y install epel-release
rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm
yum install php70w

    安装php-fpm

    yum -y install php70w-fpm
systemctl start php-fpm #启动php-fpm  
systemctl enable php-fpm #设置开机启动

    更改nginx配置文件识别php

    这里注意fastcgi_param 后面的参数，script需要改成自己的网站所在根目录

    vi /etc/nginx/conf.d/default.conf 
#把之前的#给去掉就可以了,顺手改一下fastcgi_param,向下面一样
location ~ \.php$ {  
    root           html;  
    fastcgi_pass   127.0.0.1:9000;  
    fastcgi_index  index.php;  
    fastcgi_param  SCRIPT_FILENAME  /usr/share/nginx/html$fastcgi_script_name;  
    include        fastcgi_params;  
}
    • 重启nginx和php-fpm服务
    • 在/usr/share/nginx/html/文件加下新建test.php文件输出phpinfo()内容
    • 访问http://ip/test.php即可看到nginx中的php页面
